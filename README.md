# 🚀 AI Career Readiness Dashboard

An AI-powered full-stack web application designed to help students prepare for placements with personalized roadmaps, skill gap analysis, resume feedback, and coding progress tracking.


## 🔥 Features

- 🎯 Company-Specific Roadmaps (Amazon, TCS, Deloitte)
- 📊 Skill Gap Analysis based on user profile
- 🧠 AI Resume Analyzer
- 💻 Coding Progress Tracker (LeetCode, CodeChef, Codeforces)
- 📅 Weekly Learning Plan with progress tracking
- 🔐 Authentication using Supabase
- ⚡ Fast and dynamic UI with React


## 🛠️ Tech Stack

### Frontend
- React (Vite)
- TypeScript
- Tailwind CSS

### Backend
- FastAPI (Python)

### Database & Auth
- Supabase


## 📌 How It Works

1. User signs up and selects target company
2. System generates personalized roadmap
3. Tracks weekly progress and coding scores
4. Provides resume feedback and skill gap insights
5. Continuously updates career readiness status
